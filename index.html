Summary: Latitude vs. X

The purpose of this project was to analyze weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from OpenWeatherMapAPI and get the temperatures of 500 cities.

After assembling the data, we used Matplotlib to plot different comparisons, such as Windspeed, Cloudliness, and humidity. This site provides a descriptive analysis, trends and correlation witnessed.

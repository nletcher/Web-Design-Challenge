Tesla Stock Price Predictions Using Supervised Learning Methods With Comparative Analysis of Machine Learning Models

In this project, we have implemented Recurrent Neural Network (RNN) along with Long Short-Term Memory Cells (LSTM) for Stock Market Prediction used for Portfolio Management. We used historical stock data for past 10 years from Yahoo Finance and Kaggle dataset. We have also compared the model with the traditional Machine Learning Algorithms such as Linear Regression, K-Nearest Neighbor (KNN) and Support Vector Machine (SVM) with LSTM. Our analysis results (metrics) suggest that LSTM RNN Model performed better compared to other models.

The aim of this project is to get accurate prediction and data visualization based on real data in a volatile Market environment. With the blessing of new artificial intelligence tools in machine learning, predictions are possible. This might help inverstors to decide when and what stock to invest, buy and sell to maximize profits.
